```bash
msfconsole      Run metasploit.
msfdb       Run MS DB(Database)
msfdb init      Adjustment DB for metasploit
db_status       Check MS DB status
search      Using for search module
info or advanced        Get info about module
show        It is used to list the modules and settings within the module.
use     Use module
options     It is showing settings to the module
set     Adjuts the modules settings
back        Exit module
check       Checking
```

MS using postgresql for running:
```bash
sudo systemctl enable postgresql # enable all times
sudo systemctl start postgresql # start currently
service postgresql status # check
```
